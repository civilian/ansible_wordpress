ABOUT THEME
The Dylan is a fully responsive theme for bloggers and story tellers. The theme features an optional
full size homepage slider and makes great use of featured images for posts and pages. You can also
control the theme color scheme and font choices through the customizer with no coding required.

For free themes support, please contact us https://www.electricwp.com/contact/
